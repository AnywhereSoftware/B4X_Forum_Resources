***TD\_BeeperDeLuxe***  
Features:  

- sound a beep tone
- vibrate the phone
- sound and vibrate
- sound intermitting alarmtone
- ****play own soundfile****
- ****play internal alarm sound files (included in b4xlib)****

![](https://www.b4x.com/android/forum/attachments/162292)  
  
To use you need to **implement libs as followes**:  

- core
- xui
- phone
- audio
