    • Format As String
      Gets or sets the input mask.
      Mask characters:
      # = Digit
      L = Letter
      A = Alphanumeric
      H = Hexadecimal
      ? = Any character
      Examples:
      - Date & time = "##/##/#### ##:##"
      - Phone number = "(###) ###-####"
      - IP address = "###.###.###.###"
      - MAC address = "HH:HH:HH:HH:HH:HH"
      - Parental lock password = "AAAA"
