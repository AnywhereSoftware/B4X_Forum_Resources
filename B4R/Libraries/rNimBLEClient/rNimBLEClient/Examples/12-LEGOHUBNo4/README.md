# README - rNimBLEClient LEGO HUB

Various examples using the LEGO City Hub.

LEGO Battery Box Powered Up Bluetooth HUB NO.4 (Hub 88009). 
Color & Distance Sensor (88007 / 45605).
BLE adertised name is HUB NO.4.
Handle notifications with the event OnNewData.

- LEGO® Powered Up 88009 Hub to power and control Powered Up sensors and motors.
- 2 input/output ports and a connection point for LEGO® Powered Up components.
- Connect via Bluetooth® from the LEGO® Powered Up app.
- Requires batteries.
