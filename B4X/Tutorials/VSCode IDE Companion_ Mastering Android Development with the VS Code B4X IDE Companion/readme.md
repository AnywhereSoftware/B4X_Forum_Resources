If you are a fan of the VS Code ecosystem but want to maintain the power and simplicity of **B4X** for your Android development, the **VSCode B4X Companion** extension is the bridge you’ve been waiting for.  
  
This extension allows you to handle the entire lifecycle of your B4A projects directly from within VS Code. Key features include:  

- **Integrated Emulator Control:** Start your Android emulator directly via the command palette.
- **Template-Based Projects:** Quickly spin up new projects using B4X templates without manual configuration.
- **One-Click Build & Install:** Use a single command to compile your source code and deploy it to any connected device or emulator.
- **Full Workspace Integration:** Explore and update your source code with all the VS Code features you love while the companion handles the heavy lifting.

**Simple Usage Guide:**  
  
To start a new project using the companion, follow these steps in VS Code:  

1. Open the **Command Palette** (Ctrl+Shift+P or Cmd+Shift+P).
2. Search for and select: B4X Companion: New B4X Project from Template.
3. Choose your desired template and enter a project name (no spaces!).
4. To run your app, simply select B4X Companion: Build and Install Project.
