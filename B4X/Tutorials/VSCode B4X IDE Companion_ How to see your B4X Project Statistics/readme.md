The latest version of the Visual Studio Code [B4X IDE Companion](https://marketplace.visualstudio.com/items?itemName=AneleMbangaMashy.b4x-intellisense) has functionality for one to see their Project Statistics. You can find this is Visual Studio Code.  
  
![](https://www.b4x.com/android/forum/attachments/171752)  
  
![](https://www.b4x.com/android/forum/attachments/171753)  
  
**Step 1**  
  
Open B4X Project (the b4a, b4j, b4i, b4r project is loaded. The workspace should show your project folders.  
  
**Step 2**  
  
Tap Project Statistics (this shows you some stats about your project, total lines, code lines, subs, comments, blank lines) and also some charts (top 10)  
  
Part of the statistics are broken down in table format like below.  
  
![](https://www.b4x.com/android/forum/attachments/171754)  
  
The stats shown here are for the [B4X Daisy UIKit](https://www.b4x.com/android/forum/threads/b4x-b4a-b4xdaisy-ui-kit-native-components-inspired-by-daisyui-tailwind.170352/), and its demo examples, an Android UI kit I have been working on based on DaisyUI. Our latest version for web based applications using BANano is [SithasoDaisy5](https://www.b4x.com/android/forum/threads/web-sithasodaisy5-create-websites-webapps-with-the-power-of-the-abstract-designer-opensource.165095/).
