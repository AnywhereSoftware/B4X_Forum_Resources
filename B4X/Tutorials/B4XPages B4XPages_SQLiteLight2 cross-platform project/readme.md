###  B4XPages B4XPages_SQLiteLight2 cross-platform project by klaus
### 12/09/2020
[B4X Forum - B4X - Tutorials](https://www.b4x.com/android/forum/threads/119258/)

It uses as much as possible XUI objects and common code.
