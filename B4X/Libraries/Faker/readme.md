Code example:  

```B4X

fk.Initialize  
Log(fk.FullName)     ' e.g. Alice Miller  
Log(fk.Email)        ' e.g. alice.smith@mail.com  
Log(fk.PhoneNumber)  ' e.g. +6019-8745632  
Log(fk.Address)      ' e.g. 123 Johnson St, Tomtown
```
