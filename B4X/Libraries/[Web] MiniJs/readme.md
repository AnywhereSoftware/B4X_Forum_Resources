Generate Js with B4X code.  
  
GitHub: <https://github.com/pyhoon/MiniJs-B4X>  
  
Example:  

```B4X
script1.AddFunctionCall("showSuccess", Array As String("Product created successfully!"))
```

  
Output:  

```B4X
showSuccess('Product created successfully!');
```
