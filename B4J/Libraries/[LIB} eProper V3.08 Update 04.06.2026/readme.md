**eProper v.3.08** Update 04.06.2026 - Is a helpful library to use  
with the correct spelling of words, taking into account personal exceptions.  
  
An example of a personal exception may include, among others: word: **SQLite**  
As you have noticed, this word contains upper and lower case letters,  
so no matter how you write this word, the library will always change this word  
exactly in the form: **SQLite**.  
  
To understand the essence of the library's operation, it is worth looking at the .lst files  
You will find them in the **e-Files** directory.
