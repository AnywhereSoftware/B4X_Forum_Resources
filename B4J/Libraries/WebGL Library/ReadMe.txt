Welcome to B4J jWebGL library that act as bridge for threejs Javascript library.

To get the B4J jWebGL library, you have to unzip the jWebGL.jar and jWebGL.xml
and 3 b4xtemplate files in your B4J AdditionalLibraries.

To get the library just refresh the library tab.
To get templates you need to restart the IDE.

To get more infos on how to install the library and 
to get the library version history, please refer the B4X Forum at
https://www.b4x.com/android/forum/threads/b4j-webgl-library.164553/
