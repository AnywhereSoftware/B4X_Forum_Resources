Source code is attached. You can experiment with other prompts and settings.  
Note that the format of the "state export" from the IDE is likely to change with new versions of the IDE.  
  
You can create a modified version of code bundle and call it with a command such as:  

```B4X
#Macro: Title, Export state, ide://run?File=%B4X%\\CodeBundle2.jar&Args=%STATE1%&autosave=True
```
